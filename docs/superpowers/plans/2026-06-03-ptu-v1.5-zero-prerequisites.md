Its red lines override convenience features. Do not implement account automation, broad data collection, bypasses, cookie export, or DouyinCrawler-style full crawling.

**Goal:** Release Ptu v1.5.0 as a beginner-friendly Windows desktop tool that requires no manual prerequisites for normal use.

**Architecture:** Keep the current PyInstaller onedir + Inno Setup single installer approach. Bundle required runtime binaries inside the installer, prefer bundled dependencies at runtime, and keep automatic download only as a fallback for damaged or missing bundled files.

**Tech Stack:** Python 3.13, FastAPI, pywebview, Playwright Chromium headless shell, FFmpeg, PyInstaller, Inno Setup, pytest.

```python
import ast
from pathlib import Path


def test_js_api_does_not_spawn_powershell_for_get_clipboard():
    tree = ast.parse(Path("backend/app/js_api.py").read_text(encoding="utf-8"))

    for node in ast.walk(tree):
        if isinstance(node, ast.FunctionDef) and node.name == "get_clipboard":
            source = ast.get_source_segment(
                Path("backend/app/js_api.py").read_text(encoding="utf-8"), node
            )
            assert "powershell" not in source.lower()
            return

    raise AssertionError("get_clipboard not found")
```

- [ ] **Step 3: Remove PowerShell fallback from `get_clipboard()`**

In `backend/app/js_api.py`, make `get_clipboard()` return an empty string if Win32 clipboard API cannot read text. The front end already falls back to `navigator.clipboard.readText()` and finally tells the user to press Ctrl+V.

Expected shape:

```python
    def get_clipboard(self) -> str:
        try:
            CF_UNICODETEXT = 13
            user32 = ctypes.windll.user32
            kernel32 = ctypes.windll.kernel32
            if user32.OpenClipboard(None):
                try:
                    handle = user32.GetClipboardData(CF_UNICODETEXT)
                    if handle:
                        kernel32.GlobalLock.restype = ctypes.c_void_p
                        ptr = kernel32.GlobalLock(handle)
                        if ptr:
                            try:
                                text = ctypes.wstring_at(ptr)
                                if text:
                                    return text.strip()
                            finally:
                                kernel32.GlobalUnlock(handle)
                finally:
                    user32.CloseClipboard()
        except Exception:
            pass
        return ""
```
